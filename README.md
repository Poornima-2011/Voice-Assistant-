# 🎙️ Voice Assistant Using Python

A smart desktop voice assistant built with Python that listens to user commands and performs useful tasks like opening websites, telling time/date, searching Wikipedia, telling jokes, and more through voice interaction.

## 🚀 Features

- 🎤 Speech recognition using microphone input
- 🗣️ Text-to-speech responses
- 🌐 Open websites like Google, YouTube, Gmail
- 📚 Search Wikipedia and speak summaries
- 😂 Tell random jokes
- 🕒 Tell current time and date
- 🔍 Perform web searches
- ❌ Exit on voice command

## 🛠️ Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- PyAudio
- Wikipedia
- pyjokes
- webbrowser
- datetime
